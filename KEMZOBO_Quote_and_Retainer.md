## Project Overview

Custom e-commerce website for **KEMZOBO, THE ORIGINAL ZOBO DRINK** — a premium lifestyle beverage brand. The platform enables direct-to-consumer online sales, wholesale/bulk ordering, and full business management through an admin dashboard.

| Feature | Details |
|---------|---------|

| **Newsletter/Email Capture** | Subscriber database + signup form |

| **Stripe Payments** | Payment intent creation, webhooks, auto order status updates |
| **SEO** | Meta tags, JSON-LD structured data, semantic HTML |
| **Responsive Design** | Mobile-first, optimized for Instagram/WhatsApp traffic |
| **Brand Design System** | Custom Tailwind theme, Playfair Display + DM Sans typography, micro-interactions |
