## 1. Project Description

The Developer agrees to design, develop, and deliver a custom e-commerce website for KEMZOBO, a premium lifestyle hibiscus beverage brand, hosted at kemzobo.com.

### Scope of Work Includes:

- **23 custom pages** (Home, Shop, Product Detail, About, Gallery, FAQ, Contact, Wholesale, Privacy Policy, Cart, Checkout, Order Confirmation, Login, Register, Forgot/Reset Password, My Account, Order Detail, Admin Dashboard, Admin Products, Admin Orders, Admin Wholesale, Admin Customers, Admin Messages)
- **E-commerce functionality** — shopping cart, Stripe payment processing, order management
- **Product variant system** — Single Can, 6-Pack, Case of 24 with independent pricing
- **Dynamic bulk pricing** — quantity-based discount tiers
- **Inventory management** — per-variant stock tracking with low-stock alerts
- **Wholesale inquiry portal** — B2B form with admin pipeline
- **Authentication system** — email/password login, registration, password reset
- **Email system** — order confirmations, password resets via Brevo
- **Newsletter/email capture** — subscriber database and signup forms
- **Admin dashboard** — full management of products, orders, wholesale, customers, messages
- **Custom brand design** — aligned with JEHOFILMS campaign deck and client website brief
- **Mobile-first responsive design** — optimized for social media traffic
- **SEO setup** — meta tags, structured data, semantic HTML
- **1 year free hosting** (cloud-hosted)
- **1 year free domain** (kemzobo.com)
- **SSL certificate** included
